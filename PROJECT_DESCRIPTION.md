# Descrição curta

CodeMind é uma aplicação em Python que analisa código Python e traduz a lógica encontrada para linguagem humana. O projeto usa `ast` para ler a estrutura real do código sem o executar, detetando funções, classes, ciclos, condições, imports e chamadas sensíveis. Também possui interface gráfica, modo iniciante/técnico e voz opcional com `pyttsx3`.
